# Example Web Game

A playable web game demonstrating the Voltronussy engine with canvas rendering.

## How to Play

1. Run: `pnpm game example-web`
2. Use **WASD** or **Arrow Keys** to move the player (green square)
3. Collect the yellow collectibles
4. Avoid the red enemies!

## Controls

| Key | Action |
|-----|--------|
| W / ↑ | Move up |
| S / ↓ | Move down |
| A / ← | Move left |
| D / → | Move right |

## Features Demonstrated

- Entity-Component-System architecture
- Keyboard input handling
- Collision detection (simple AABB)
- Canvas rendering
- Fixed timestep physics
